# Café da Esquina - Site Institucional

Este é o MVP (Produto Mínimo Viável) do site institucional para a cafeteria **Café da Esquina**. O objetivo é divulgar o negócio na internet, apresentando a cafeteria e suas principais informações de forma simples, bonita e responsiva.

## 🎯 Objetivo do Projeto

Criar um site institucional responsivo que apresente a cafeteria e suas principais informações, seguindo a identidade visual e os requisitos definidos pelo cliente.

## ✅ Funcionalidades do MVP

O site inclui as seguintes seções:

*   **Header:** Com o nome da marca e menu de navegação.
*   **Seção Principal de Destaque (Hero):** Apresentando a proposta da cafeteria.
*   **Seção Sobre a Cafeteria:** Detalhes sobre a história e o conceito do Café da Esquina.
*   **Seção de Produtos/Cardápio:** Destaques do cardápio base.
*   **Seção com Horário e Localização:** Informações essenciais para os clientes.
*   **Seção de Contato:** Formulário simples para comunicação.
*   **Footer:** Com informações de direitos autorais e links para redes sociais.

## 📱 Requisitos Obrigatórios

*   **Responsividade:** O site se adapta a diferentes tamanhos de tela, funcionando bem em dispositivos móveis.
*   **HTML Semântico:** Utilização de tags HTML apropriadas para estruturar o conteúdo.
*   **Organização Visual:** Design limpo, atraente e com boa hierarquia visual.
*   **Imagens:** Uso de imagens relevantes para enriquecer a experiência do usuário.
*   **Navegação Clara:** Menu intuitivo e fácil de usar.
