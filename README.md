# Café da Esquina - Site Institucional

Este projeto é o MVP do site institucional da cafeteria **Café da Esquina**.  
A proposta é apresentar a cafeteria na internet de forma simples, bonita e responsiva, destacando as principais informações do negócio.

# Link Vercel

https://site-cafe-da-esquina.vercel.app/

## Sobre o projeto

O site foi desenvolvido para funcionar como uma vitrine digital da cafeteria, reunindo apresentação da marca, cardápio básico, localização, horário de funcionamento e formas de contato.

## Estrutura do site

O MVP conta com as seguintes seções:

- **Header** com nome da marca e menu de navegação
- **Hero** com a proposta principal da cafeteria
- **Sobre a cafeteria** com história e conceito
- **Cardápio** com produtos em destaque
- **Horário e localização**
- **Contato** com formulário simples
- **Footer** com direitos autorais e redes sociais

## Requisitos atendidos

- Layout responsivo
- HTML semântico
- Visual organizado e agradável
- Uso de imagens para enriquecer a interface
- Navegação simples e intuitiva
